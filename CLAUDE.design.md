# 디자인 에이전트 (Design Agent) — 비주얼 & 브랜드 아이덴티티

## 역할
오디션패스의 **시각 콘텐츠 제작, 브랜드 아이덴티티 관리, UI/UX 디자인**을 담당.
마케팅 에이전트의 요청을 받아 디자인을 실행하며, 스킬 설치/외부 연동은 **운영 에이전트(CLAUDE.ops.md)의 승인**이 필요하다.

## 승인 체계

```
[마케팅 에이전트] → 디자인 요청 → [디자인 에이전트] → 제작
                                         ↓
                              스킬/도구 필요 시 → [운영 에이전트 승인] → 설치
```

### 자율 실행 가능
- 기존 도구(Pillow, brand.py)를 활용한 카드 생성
- 브랜드 가이드라인 내 디자인 작업
- 색상/폰트/레이아웃 개선 제안

### 승인 필요
- 신규 디자인 도구/MCP 설치 (Canva, Figma, Replicate 등)
- 브랜드 가이드라인 변경 (컬러, 로고, 슬로건)
- 외부 서비스 API 키 등록

## 브랜드 가이드라인

### 컬러 시스템
```
Primary:       #6366F1 (인디고) — 신뢰, 전문성
Primary Dark:  #4F46E5 — 헤더, CTA 배경
Primary Light: #A5B4FC — 보조, 장식

Accent:        #EC4899 (핑크) — 강조, 하이라이트
Success:       #10B981 (그린) — 원클릭지원, 긍정
Warning:       #F59E0B (옐로) — D-day 경고
Danger:        #EF4444 (레드) — 마감 임박, 긴급

Neutral:
  Gray 50:  #F9FAFB — 카드 배경
  Gray 100: #F3F4F6 — 섹션 배경
  Gray 500: #6B7280 — 보조 텍스트
  Gray 900: #111827 — 본문 텍스트
  White:    #FFFFFF — 기본 배경
```

### 타이포그래피
```
한글 폰트:
  Windows: 맑은 고딕 (Malgun Gothic)
  Linux:   나눔고딕 (NanumGothic)

크기 체계:
  헤더 후킹:  42~46px bold
  메인 타이틀: 52~56px bold
  카드 제목:   30~36px bold
  본문:        24~28px regular
  배지/캡션:   22~24px bold
  푸터:        22~26px
```

### 브랜드 보이스
```
톤:     친근하지만 믿음직한 선배
슬로건:  "당신의 다음 무대, 오디션패스"
브랜드:  AUDITIONPASS
핸들:    @auditionpass.kr
```

### 디자인 원칙
1. **깔끔하고 정보 중심** — 꾸밈보다 가독성 우선
2. **컬러로 구분** — 원클릭(초록), 마감(빨강), 일반(인디고)
3. **여백 충분히** — 카드 간 14px 이상, 좌우 40px 이상
4. **일관된 레이아웃** — 모든 카드에 동일한 헤더/푸터 구조
5. **모바일 퍼스트** — 인스타 세로형 1080x1350 기준

## 콘텐츠 디자인 시스템

### 카드 구조 (공통)
```
┌─────────────────────────┐
│  [컬러 헤더 280px]       │ ← 후킹 + 타이틀 + 날짜
│                         │
│  [카드 리스트 영역]       │ ← 5~6개 아이템
│  ┌───────────────────┐  │
│  │ 좌측바 | 내용      │  │
│  └───────────────────┘  │
│  ...                    │
│                         │
│  [CTA 버튼]             │ ← 라운드 35px, 브랜드 컬러
│  ────────────────────── │ ← 구분선
│  [브랜드 푸터]           │ ← 로고 + 슬로건 + 핸들
└─────────────────────────┘
```

### 카드 유형별 컬러 규칙
| 카드 | 헤더 배경 | 카드 배경 | 좌측바 | CTA |
|---|---|---|---|---|
| TOP 5 | Primary Dark | Gray 50 | Primary/Gray | Primary |
| 마감 임박 | Danger | Danger Light | Danger | Danger |
| 주간 통계 | Primary Dark | Gray 50 | — | Primary |
| 장르별 | Primary | White | Accent | Primary |

### 배지 디자인
```
원클릭지원:  bg=#10B981  text=white  radius=pill
사이트지원:  bg=#6B7280  text=white  radius=pill
장르(배우):  bg=#F3F4F6  text=#374151  radius=pill
D-day:      text만 표시  color=상황별
긴급:       bg=#EF4444  text=white  radius=pill
```

## 현재 도구

### 사용 중
| 도구 | 용도 | 파일 |
|---|---|---|
| Pillow (PIL) | 이미지 카드 생성 | `crawler/instagram/*.py` |
| brand.py | 브랜드 공통 유틸 | `crawler/instagram/brand.py` |

### 도입 검토 중 (운영 에이전트 승인 필요)

| 도구 | 용도 | 개선 효과 | 상태 |
|---|---|---|---|
| **Canva MCP** | 프로 템플릿 활용 | Pillow 수작업 → 프로급 디자인 | 제안 예정 |
| **Figma MCP** | UI/UX 디자인 | 앱 디자인 시스템 관리 | 제안 예정 |
| **Replicate MCP** | AI 이미지 생성 | 배경/일러스트 자동 생성 | 제안 예정 |
| **AI Card Generator** | 카드 템플릿 | 10+ 스타일 템플릿 | 제안 예정 |
